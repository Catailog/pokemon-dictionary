# pokemon-dictionary

## 프로젝트 소개

외부 공개 API와의 통신을 통해 데이터를 조회하고, 비동기 처리 및 데이터 가공 과정을 연습하기 위해 만든 개인 프로젝트이다.

## 주요 기능

- 이전/다음 ID의 포켓몬 데이터 조회
- 포켓몬 ID를 기반으로 한 데이터 조회

## Third-Party Assets

### Data Source

[PokeAPI](https://pokeapi.co/): 모든 포켓몬 데이터는 PokeAPI를 통해 제공받았습니다.
